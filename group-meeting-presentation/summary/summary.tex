\section{序列相关的基本性质汇总(预测已经见过的任务)}

\begin{itemize}
  \item 预测长度越短，预测结果的误差就越小.
  \item 对于相同的训练任务，给定的训练数据较多时，预测结果的误差就越小。
  % \item 对于一个预测长度，预测函数相同的模型，训练数据超出一定的阈值的时候，会使得误差变大，猜测可能是因为过拟合的原因导致的。
  \item 对于相同的训练任务，给定的训练数据超过某个阈值，会使得预测结果的误差变大。(这可能是因为过拟合导致的)
  % \item 有不同预测长度的序列模型的预测结果良好。(不太好)
  % \item 当预测任务，以及预测序列相同时，可以看出给定训练的序列越长，效果越好。
  \item 进行预测时,历史序列的长度越长,最终预测结果的误差就越小.
  % \item 当训练数据集的数量，测试的序列相同时，预测的长度越短，效果越好。
  \item 进行预测时,需要预测的长度越短,最终预测结果的误差就越小.
  % \item 当训练总数据集数量，预测长度相同时，训练的序列向量越长，对最终的预测结果更加精确。
  \item 对于两种不同的函数序列(其中预测维度的函数相同),那么其中维度更高的函数序列,最终的预测误差会更小.
  % \item 对于一个预测长度不一致的数据集合，最终的误差比较大，且预测序列长度和目标序列长度相似度大约是76\%
  \item 在多种类函数序列训练任务中,预测结果的误差,以及任务的通过率均低于单一训练任务.
  % \item 从混合的预测长度，时间序列的数据集合训练结果中也可以看出这个长向量时间序列的预测结果要好于短向量的结果，且预测短的序列要优于预测长序列的误差。
  \item 在对于多种预测长度的训练任务中,同样可以看到历史序列的长度越长以及需要预测的长度越短,最终预测结果的误差就越小.
  % \item 从中可以看出，不同的预测长度对于精确度影响不大，但是不同种的序列对精确度的影响比较大。
  \item 不同的预测长度的训练任务对于预测结果精确度影响不大，不同的函数序列训练任务对精确度的影响比较大。
  % \item 但类型数量取1000的时候，对于不同种类的序列会使得最终误差较大，但是提高数量到5000，能够让误差明显的变小。
  \item 同上,通过增加训练数据的数量,可以改善预测结果的误差.
\end{itemize}

\section{泛化性结论汇总(未见过的预测任务)}
\begin{itemize}
  \item 预测的函数序列和训练的函数序列相同,但预测序列的长度和训练的长度不同,,结果发现正确的预测长度应该是4,但是结果预测长度全部是8.从中可以看出,模型在解决预测长度不同的新问题上泛化能力较弱.
  \item 在预测没见过的任务上,误差明显大于见过的任务.
  \item 可能预测序列的维度对预测结果的误差影响不大.
  \item 提高训练的数据量,对于最终的通过率有明显的提升.但是预测的准确率没有明显的上升.
\end{itemize}

\section{从相关论文中总结出的一些结论}
\begin{itemize}
  \item 在使用的10个语言模型和10个时间序列模型的比较中
  \eee{
    \item 大部分语言模型的性能是要高于时间序列模型。在三个测试数据集中，大部分语言模型的MAE和MSE均超越了时间序列预测模型。
    \item 测试的几个的预训练模型中,Pretrained Model Size越大,预测效果也越好.
  }
  \item 关于模型泛化能力的测试：
  \eee{
    \item 在泛化能力的测试上面，语言模型的预测结果的误差小于时间序列模型。说明语言模型的泛化能力更强.
    % \item 泛化的任务对于语言模型而言，肯定也不如非泛化的效果。
    \item 语言模型在处理泛化任务上,也能够保证一定的缺失率(通过率)。
  }
  \item 关于序列预测长度的相关结论
  \eee{
    \item 给定同样的历史序列长度，预测长度越短，预测结果误差越小
    \item 预测相同的长度，给定的历史序列越长，预测结果误差越小
  }
  \item 提示指令的相关结论
  \eee{
    \item 关于提问，相互间隔有规律的提问方式(类似表格形式的提问方式)比并排(简单的陈述句)的方式在预测效果上更好。
    \item 通过热力图说明transformer的多头注意力机制是如何解读指令数据的内容.
    \item 对于语言模型而言,他能够将注意力的大部分比重都放在有效提示中(提示命令中的序列语句),在无效的提示词中有更小的比重.
    \item 语言模型确实用到了历史序列的信息，这为语言模型进行时间序列的预测奠定基础.
    % 具体的提问语言模型是如何进行预测未来的数据的，他能够使用transformer的注意力来进行预测！
    % \item 通过使用热力图，将提示中的每一个单词的重要性都展示出来了，可以看到越到后面几层，提示中的历史数据就变成最重要的信息了，这不仅说明了这个提示序列的重要性，还
  }
\end{itemize}
